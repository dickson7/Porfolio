Ejercicios con JavaScript

Galeria de fotos

Una hermosa galeria receptiva, util para muchos proyectos. Se utiliza magnific-popup.css cdnjs. La galeria de fotos es responsiva.

Formulario Animado

Este ejercicio es un formulario animado, donde el background cambia de color cuando los datos son incorrectos. Tambien se realiza validacion de correo para que contenga el @ seguido del dominio.

Transisión de paginas

Este ejercicio realizamos transiciones entre paginas utilizando Barba.js una biblioteca excelente para usar en sitios estaticos y hacer unas transiciones geniales, en lugar de incluir un marco completo con React.

Efecto de escritura

Con solo 20 lineas de codigo JavaScript, logramos el efecto de escritura. Muy sencillo y util.

Seccion testimoniales

Un sección util para historias o testimonios de usuarios o clientes, se puede utilizar en cualquier proyecto. Se utiliza OwlCarousel2.css cdnjs. Los testimoniales de clientes es responsiva.

Scroll animado

Ejercicio con animacion del scroll. Se utiliza scrollmagic cdnjs. Este efecto fue tendecia en 2017 para en paginas muy largas darle un estilo mas atractivo y divertido.

Parallax

Ejercicio parallax, un efecto que al entrar en una web y hacer scroll el fondo parece moverse a una velocidad distinta que el contenido.

Menu Hambuerguesa

Un menu Hambuerguesa con algunas transisiones en JavaScript. Animaciones en el boton de activación y despliege de los links.
